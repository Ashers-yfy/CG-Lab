## 实现说明

### 基础任务

**迭代式光线弹射**：用 for 循环替代递归，最多弹射 N 次（可调）。每次击中镜面则更新方向继续追踪，击中漫反射则着色后终止。

**硬阴影**：从交点沿光源方向发射暗影射线，若在到达光源前击中物体则判定为阴影，只计算环境光。

**Shadow Acne 避坑**：暗影射线起点沿法线偏移 1e-4，避免自相交产生黑色噪点。

**棋盘格纹理**：通过交点 x、z 坐标取整后奇偶性判断格子颜色。

### 选做1：玻璃折射

基于斯涅尔定律（Snell's Law）计算折射方向，折射率 1.5。发生全反射时退化为镜面反射处理。

### 选做2：MSAA 抗锯齿

每个像素内随机发射多条主射线（1~16 条可调），对颜色取平均，消除物体边缘锯齿。
