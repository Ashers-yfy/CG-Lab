# 实验2：旋转与变换

## 实验简介

基于 Taichi 实现三维 MVP（Model-View-Projection）坐标变换，将三维顶点投影到二维屏幕并绘制线框图形。

## 操作说明

| 按键 | 功能 |
|------|------|
| `A` | 逆时针旋转 |
| `D` | 顺时针旋转 |
| `T` | 切换三角形 / 立方体 |
| `ESC` | 退出 |

```bash
pip install taichi

```

## 实现说明

### get_model_matrix — 绕 Z 轴旋转矩阵

角度转弧度后构造标准 Z 轴旋转矩阵。

### get_view_matrix — 视图矩阵

将相机从 eye_pos 平移至原点，相机朝向 -Z 方向。

### get_projection_matrix — 透视投影矩阵

分两步：
1. M_persp→ortho：将视锥体压缩为正交长方体
2. M_ortho：平移+缩放归一化到 NDC [-1,1]³

右手坐标系下 n = -zNear，f = -zFar。

### 选做：三维立方体

- 8 个顶点，坐标范围 [-1, 1]，中心在原点，边长为 2
- 12 条边构成完整线框，展示透视效果的三维旋转
