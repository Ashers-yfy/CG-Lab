# CG-Lab: 交互式粒子群算法视觉展示

本项目是一个基于 **粒子群算法 (Particle Swarm Optimization, PSO)** 的视觉交互系统。通过模拟鸟群或鱼群的群体行为，实现在复杂空间中的动态路径寻找与交互反馈。

## 🛠 项目架构 (src 布局)
项目代码采用标准模块化布局，确保逻辑与渲染分离：
- `src/`
  - `main.py`: 项目入口，负责初始化窗口与主循环。
  - `particle_system.py`: 核心逻辑类，包含粒子的速度、位置更新算法。
  - `utils.py`: 辅助工具函数。
- `.gitignore`: 配置文件，已忽略 `.venv`、`__pycache__` 等本地环境。
- `README.md`: 项目说明文档。

## 🧠 算法逻辑介绍

### 1. 核心原理
粒子群算法模拟了群体智能的决策过程。每个“粒子”在搜索空间中通过三个分量来决定下一步的运动方向：
* **惯性 (Inertia)**：粒子维持原有运动状态的趋势。
* **个体认知 (Cognitive)**：粒子向其自身历史最优位置 ($P_{best}$) 靠近。
* **社会经验 (Social)**：粒子向整个群体发现的全局最优位置 ($G_{best}$) 靠近。



### 2. 数学模型
在代码实现中，粒子 $i$ 的速度 $v$ 和位置 $x$ 更新公式遵循：
$$v_{i}^{t+1} = w v_{i}^{t} + c_1 r_1 (P_{best,i} - x_{i}^{t}) + c_2 r_2 (G_{best} - x_{i}^{t})$$
$$x_{i}^{t+1} = x_{i}^{t} + v_{i}^{t+1}$$
